# Agent Notes

## DAP Data Integrity

- Discovered products are candidates only; never add a DAP automatically.
- Do not modify `src/data/daps.csv` unless the user explicitly asks for a data update.
- Verify official sources first and keep source-backed notes specific.
- Distinguish model identity from variant, color, finish, bundle, region, and storage-only differences.
- Do not collapse generations, editions, or revisions without source evidence.
- Exclude non-DAP portable audio products such as dongle DACs, Bluetooth receivers, headphone amplifiers, IEMs, headphones, cables, cases, and accessories.
- AI-assisted work must not invent unsupported specs.
